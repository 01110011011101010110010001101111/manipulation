Category-Level Manipulation

What is the right representation for an object?

So far our answer to this question has been pretty narrow. We've basically asked for a sdf file..

The manipulation tasks that we've considered so far fall into one of two cases. In the first case, if we have a small number of objects that we wish to manipulate, and we have models for those objects, then we have tools to find them (e.g. via instance segmentation) in a scene, estimate their pose, plan a grasp, etc. We could use this pipeline to perform a variety of manipulation tasks, but the assumption that we have model is quite limiting. And it's nontrivial to get models that are sufficient for both simulation and rendering. The second case we have considered was in the bin picking / clutter clearing example, where we can pick up almost any object, with no model required, but the tasks we can accomplish without any object model are quite limited. So far we've only dropped things into the other bin.
